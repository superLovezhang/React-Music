## 网易云音乐移动端(React)
* 技术栈选型: react + react-router-dom + redux + redux-thunk + swiper
* 使用：
    1. git clone
    2. yarn install
    3. yarn start 
    4. 建议在手机或者浏览器的模拟器调试状态使用
